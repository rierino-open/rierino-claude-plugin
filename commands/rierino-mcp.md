# Rierino MCP Lists

The user wants a quick overview of available tools, resources, prompts or templates from the Rierino platform.

1. Call 'tools/list', 'resources/list', 'resources/templates/list' and 'prompts/list' methods on rierino MCP server.
2. List results as a table, do not make a call to any of the listed tools unless user has requested it explicitly.
